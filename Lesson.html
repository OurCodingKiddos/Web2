Canvas Playground (HTML Only)

The <canvas> tag creates a blank drawing area. Later we will use JavaScript to draw shapes, characters, or games. For now, we are just placing it on the page.

Right now, it is empty because drawing requires JavaScript. But it's a good start to understand how the canvas is placed on the page.

Key Points to Remember

  • <canvas> creates a drawing area
  • width controls how wide it is
  • height controls how tall it is
  • It does nothing yet â€” drawing comes later with JavaScript

Try This!

Change the width and height numbers and refresh the page to see how the canvas size changes.

<canvas id="myCanvas" width="300" height="150"></canvas>

Explore the World with Iframes

The <iframe> tag lets us embed another webpage, video, or map inside our own page. Think of it as a magic window to another world!

Example 1: YouTube Video

Embed a video directly in your page.

Example 2: Google Map

Show a location or game studio on a map!

Example 3: Another Website

We can display another webpage inside our page, like W3Schools!

Key Points to Remember

  • <iframe> = inline frame
  • Allows embedding of videos, maps, or entire webpages
  • src = URL of the content to display
  • width & height = size of the window
  • allowfullscreen lets users watch in full screen

Great job! You've learned three new HTML concepts: figures, canvas, and iframes. Each of them adds more power and fun to your webpages.
